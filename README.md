# HoloGene AI - Pharmacogenomics Analysis Platform

![HoloGene AI](https://img.shields.io/badge/React-19-blue) ![Node.js](https://img.shields.io/badge/Node.js-Express-green) ![OpenAI](https://img.shields.io/badge/OpenAI-API-orange)

A full-stack web application for analyzing VCF (Variant Call Format) genomic data with AI-powered clinical insights.

## Features

✅ **Secure Login** - Doctor authentication (username: `doctor`, password: `1234`)  
✅ **VCF File Upload** - Upload genomic variant files or paste VCF content  
✅ **Genetic Analysis** - Parse and visualize genetic variants  
✅ **AI Summaries** - OpenAI-powered clinical interpretation of variants  
✅ **Results Dashboard** - View detailed variant information and AI analysis  
✅ **DNA Animation** - Beautiful DNA helix background UI  

## Tech Stack

### Frontend
- **React 19** - UI framework
- **React Router DOM 7** - Client-side routing
- **Axios** - HTTP client for API calls
- **Framer Motion** - Animation library
- **CSS** - Inline styling with DNA theme

### Backend
- **Node.js + Express 5** - REST API server
- **Multer 2** - File upload handling
- **CORS** - Cross-origin requests
- **OpenAI API v4+** - AI clinical summaries
- **dotenv** - Environment variable management
